# 🚀 快速开始指南

只需3步，立即启动你的AI聊天应用！

---

## ⚡ 方法一：使用启动脚本（推荐）

### Windows 用户

双击运行 `启动脚本.bat` 或在命令行中执行：

```cmd
cd docker文件
启动脚本.bat 生产
```

### Linux / macOS 用户

```bash
cd docker文件
chmod +x 启动脚本.sh
./启动脚本.sh 生产
```

---

## 📦 方法二：使用 Docker Compose

### 1. 进入项目目录
```bash
cd AI
```

### 2. 启动应用
```bash
# 生产环境
docker-compose -f docker文件/docker-compose.yml up -d

# 或开发环境（支持热重载）
docker-compose -f docker文件/docker-compose.dev.yml up -d
```

### 3. 访问应用
打开浏览器访问：**http://localhost:8000**

---

## 🛑 停止应用

### Windows
```cmd
启动脚本.bat 停止
```

### Linux / macOS
```bash
./启动脚本.sh 停止
```

### 或使用 Docker Compose
```bash
docker-compose -f docker文件/docker-compose.yml down
```

## ⚙️ 环境变量配置（可选）

如果需要配置 API Keys，在项目根目录创建 `.env` 文件：

```env
OPENAI_API_KEY=your-openai-api-key
GOOGLE_API_KEY=your-google-api-key
```

---

## 💡 提示

- 首次启动会自动下载和构建镜像，需要几分钟
- 默认端口是 8000，如果被占用可以修改 `docker-compose.yml`
- 开发环境支持代码热重载，适合本地开发
- 生产环境性能更优，适合实际部署
